# 行业应用案例

本页展示 AI 在不同行业的典型应用方向，内容以资料介绍为主。

## 典型场景

### 教育行业

- 智能答疑与作业讲解
- 课程内容自动总结
- 个性化学习建议

### 企业办公

- 会议纪要自动整理
- 报告生成与润色
- 合同与制度文档问答

### 电商与客服

- 商品文案生成
- 多轮客服辅助回复
- 售后问题分类与分派

### 医疗与健康（非诊断）

- 医疗科普内容整理
- 体检报告文本解释辅助
- 健康知识检索

## 上线前检查清单

1. 数据是否合规，敏感信息是否脱敏
2. 是否设置人工复核流程
3. 是否具备日志记录与问题追踪能力
4. 是否有误答兜底策略

## 持续优化建议

- 每周回收真实问题样本
- 按业务指标评估效果（准确率、响应时间、满意度）
- 持续迭代提示词与知识库内容
