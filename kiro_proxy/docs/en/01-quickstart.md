# AI Overview

This page introduces essential AI knowledge and a beginner-friendly learning path.

## What Is Generative AI?

Generative AI systems can produce text, images, audio, code, and other content.

Common capabilities:

- Conversational Q&A
- Summarization and rewriting
- Code generation and explanation
- Image understanding

## 30-Day Learning Path

1. Week 1: Learn core concepts (LLM, token, context window, prompt)
2. Week 2: Practice prompts for summarize/extract/rewrite tasks
3. Week 3: Understand RAG and knowledge-grounded answering
4. Week 4: Build a small project (document assistant or support bot)

## Quick Glossary

| Term | Meaning |
|---|---|
| LLM | Large Language Model |
| Prompt | Instruction given to a model |
| RAG | Retrieval-Augmented Generation |
| Embedding | Vector representation of text |
| Hallucination | Confident but incorrect model output |
