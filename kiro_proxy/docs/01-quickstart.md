# AI 入门导览

本页面用于展示人工智能基础资料与学习路径，适合初学者快速了解 AI 核心概念。

## 什么是生成式 AI

生成式 AI 是指可以生成文本、图像、音频、代码等内容的模型系统，常见能力包括：

- 对话问答
- 文本总结与改写
- 代码生成与解释
- 图片理解与内容创作

## 30 天学习路径

1. 第 1 周：理解大模型、Token、上下文窗口、提示词等基础概念
2. 第 2 周：学习提示词写法，掌握总结、抽取、改写三类任务
3. 第 3 周：了解 RAG（检索增强生成）与知识库问答
4. 第 4 周：完成一个小项目，例如 AI 文档助手或客服问答助手

## 关键词速查

| 术语 | 含义 |
|---|---|
| LLM | 大语言模型 |
| Prompt | 提示词，给模型的输入指令 |
| RAG | 检索外部资料后再生成回答 |
| Embedding | 把文本转为向量表示 |
| Hallucination | 模型“幻觉”，即不准确或编造内容 |

## 使用建议

- 优先给出明确目标、输入格式和输出格式
- 重要结论要要求模型标注依据
- 对高风险信息进行人工复核
